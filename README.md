# DevFolio  

DevFolio is a standalone **Dev**eloper Port**folio** built using Angular. This project was created for learning purposes and showcases developer skills, projects, and experience in a sleek, modern design.

## 🚀 Features  
- Fully responsive design  
- Modular and scalable architecture  
- Simple setup and easy to run  
- Built with Angular as a standalone application  

## 📦 Getting Started  

### 1. Clone the Repository  

```sh
git clone https://github.com/yourusername/devfolio.git
cd devfolio
```

### 2. Install Dependencies  

```sh
npm install
```

### 3. Run the Application  

```sh
ng serve
```

By default, the app will run on **http://localhost:4200/**. Open it in your browser to view the portfolio.
